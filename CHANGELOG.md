### Changelog

## 1.1.0 — 12.12.2025

### Added
- Добавлены предопределенные свойства processname и filename для вывода в отчет. Помогают определить, в каком конкретно процессе rphost_* и файле *.log найдено событие. Например, при анализе TTIMEOUT или TDEADLOCK.
- Добавлена команда группового изменения списка анализируемых каталогов. Позволяет сделать постоянное групповое добавление каталогов для анализа в выбранные шаблоны формы списка справочника "Шаблоны анализа технологического журнала".
- Добавлена команда группового изменения списка анализируемых каталогов. Позволяет сделать временное групповое добавление каталогов для анализа в выбранные шаблоны обработки "Анализ технологического журнала".
- В обработке "Анализ технологического журнала" добавлена команда "Заполнить отборы", позволяющая выполнить групповую установку отборов по всем шаблонам для анализа.
 
### Changed
- Обновлено руководство пользователя
- Исправлены ошибки
